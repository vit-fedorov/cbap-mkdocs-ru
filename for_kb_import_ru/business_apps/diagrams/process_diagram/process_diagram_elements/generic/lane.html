Дорожка — используется для упорядочивания элементов внутри пула в соответствии с бизнес-функциями и ролями их исполнителей.

Операции в меню элемента «Дорожка»

  • Действия
    • Свойства ‌ — переход к окну свойств дорожки.
    • Удалить ‌ — удаление данного элемента из диаграммы процесса.

Свойства дорожки

В окне свойств дорожки можно настроить общие свойства элемента диаграммы процесса, а также указать исполнителей.

Исполнители — выберите в этом поле роль, которая будет автоматически назначаться пользовательским задачам, создаваемым на данной дорожке. Название назначенной роли будет отображаться на диаграмме рядом с названием дорожки.
