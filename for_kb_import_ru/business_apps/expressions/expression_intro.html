При разработке приложений часто требуется создать продвинутую бизнес-логику, что не всегда достижимо с помощью визуальных инструментов.

Для вычислений и определения логики Comindware Platform позволяет использовать выражения на трёх языках:

  • Формулы — встроенный в платформу язык программирования, по синтаксису схожий с формулами Excel.

    Узнайте больше о языке формул Comindware в Руководстве по языку формул.

    Примеры:

    • Установка значения атрибута по условию;
    • Вычисление всех аккаунтов группы;
    • больше примеров: Примеры формул.

  • C# — объектно-ориентированный язык программирования, который используется для обращения к API Comindware Platform, API внешних систем и типовым библиотекам C#.

    Узнайте больше о написании C#-скриптов в Руководстве по написанию скриптов на языке C#.

    Примеры:

    • Копирование объекта;
    • Кнопка «Мой профиль»;
    • больше примеров: Примеры на C#.

  • N3 — это язык описания, базирующийся на триплетах вида «субъект-предикат-объект». В отличие от формул язык N3 позволяет производить более сложные вычисления.

    Узнайте больше о языке N3 Comindware в Руководстве по языку N3.

    Примеры:

    • Фильтр списка по создателю;
    • Сравнение двух атрибутов типа «Аккаунт»;
    • больше примеров: Примеры на N3.
