  • Компоненты фронтенда

  • Вспомогательное ПО для Astra Linux

Введение

Некоторые функции Comindware Business Application Platform (далее «Система») реализованы с использованием сторонних библиотек.

Для обеспечения работы Системы используются сторонние сервисы, такие как веб-сервер, СУБД, обратный прокси-сервер и т. п.

В настоящей статье представлен перечень стороннего программного обеспечения (ПО), которое поставляется вместе с Comindware Business Application Platform версии 4.7 для операционных систем (ОС) семейства Linux: «Альт Сервер» 10.0–10.1, Astra Linux SE 1.7.0 «Смоленск».

Состав стороннего ПО для других операционных систем может отличаться.

  • Вспомогательное ПО для Astra Linux

Введение

Некоторые функции Comindware Business Application Platform (далее «Система») реализованы с использованием сторонних библиотек.

Для обеспечения работы Системы используются сторонние сервисы, такие как веб-сервер, СУБД, обратный прокси-сервер и т. п.

В настоящей статье представлен перечень стороннего программного обеспечения (ПО), которое поставляется вместе с Comindware Business Application Platform версии 4.7 для операционных систем (ОС) семейства Linux: «Альт Сервер» 10.0–10.1, Astra Linux SE 1.7.0 «Смоленск».

Состав стороннего ПО для других операционных систем может отличаться.
